## Сервер
1) Зайти в sever 

2) выполнить: 

```bash
$ npm install
```

3) выполнить: 

```bash
$ npm run dev
```

## Клиент
1) Зайти в client 

2) выполнить: 

```bash
$ npm install
```

3) выполнить: 

```bash
$ npm run dev
```

## Описание
1) Открываем юрл client сервера

2) Вбиваем Field Size и Diamonds и жмем Create Game

3) Копируем юрл и параллельно открываем в другом браузере

4) Играем
